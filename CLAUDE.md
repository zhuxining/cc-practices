## ANTI-PATTERNS (❌ Don't do it)

1. **Don't edit `routeTree.gen.ts`** - automatically generated, it will be overwritten
2. **Do not edit `apps/web/src/components/ui/\*** - managed by shadcn CLI
3. **Do not skip the session check in protectedProcedure** - The middleware has been processed automatically
4. **Do not manually create migration files in the development environment** - use `db:push` instead
5. **Do not modify the table structure generated by Better-Auth** - Automatic management of authentication tables
